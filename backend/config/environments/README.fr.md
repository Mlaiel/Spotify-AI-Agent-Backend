# Module Environnements – Spotify AI Agent

## Présentation
Ce dossier contient tous les fichiers d’environnement et de configuration pour le backend. Il garantit que chaque environnement (Développement, Production, Staging, Test) est sécurisé, auditable et prêt pour le CI/CD.

### Fonctionnalités principales
- Fichiers .env complets et commentés pour chaque environnement
- Sécurité, monitoring, ML, conformité, feature-flags, logging
- Compatible Docker, Cloud, K8s, CI/CD
- Jamais de secrets réels dans le repo (seulement des exemples)

### Bonnes pratiques
- Ne jamais commiter de vrais secrets ou mots de passe !
- Utiliser .env.production uniquement avec de vrais secrets en production
- Toutes les variables sont documentées et adaptées aux rôles

### Équipe & Rôles
- Lead Dev, Architecte IA, Backend Senior, ML Engineer, DBA/Data Engineer, Security Specialist, Microservices Architect
